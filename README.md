# ML Classification Models 🤖

Machine Learning classification models built 
using Scikit-learn with professional evaluation.

## Models Built
- Decision Tree Classifier
- Random Forest Classifier (100 trees)
- Cross Validation (cv=5)
- Hyperparameter Tuning (GridSearchCV)

## What it does
- Predicts student pass/fail
- Evaluates with confusion matrix
- Precision, Recall, F1-score report
- Finds best model parameters automatically

## Tech Stack
- Python
- Scikit-learn
- Numpy
- GridSearchCV

## Results
- Decision Tree: 100% accuracy
- Cross validation: 95% mean accuracy
- Best parameters found automatically!

## Files
- classification.py → Decision Tree + Random Forest
- model_evaluation.py → Confusion matrix + reports
- cross_validation.py → CV + hyperparameter tuning

## How to run
pip install scikit-learn numpy
python classification.py
